Configuration (config.ini)

Server Settings

Key Value
debug True or False. Debug mode controls several development features
port Set any port number you please, this is only relevant if you are running the development server.
secret_key Set a secret key so cryptographic components can use this to sign cookies and other things

Site Settings

Key Value
name The name of your website. Can be called upon by views to automatically set page title, etc.
desc And just like the your website's name, the description can also be called easily through views / templates.

MySQL Settings

Key Value
host Typically left as `localhost` unless the mysql server is hosted elsewhere.
user MySQL login username
database MySQL database name
Update View update history
